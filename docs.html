Capacitor MediaStore Plugin

Complete access to Android MediaStore API and iOS Photos framework

Installation

npm install @odion-cloud/capacitor-mediastore
npx cap sync

Android Setup

Add permissions to android/app/src/main/AndroidManifest.xml:

<!-- Legacy permissions for Android 6-12 -->
<uses-permission android:name="android.permission.READ_EXTERNAL_STORAGE" 
                 android:maxSdkVersion="32" />
<uses-permission android:name="android.permission.WRITE_EXTERNAL_STORAGE" 
                 android:maxSdkVersion="29" />

<!-- Granular permissions for Android 13+ -->
<uses-permission android:name="android.permission.READ_MEDIA_IMAGES" />
<uses-permission android:name="android.permission.READ_MEDIA_AUDIO" />
<uses-permission android:name="android.permission.READ_MEDIA_VIDEO" />

<!-- Visual media permissions for Android 14+ -->
<uses-permission android:name="android.permission.READ_MEDIA_VISUAL_USER_SELECTED" />

iOS Setup

Add usage descriptions to ios/App/App/Info.plist:

<key>NSPhotoLibraryUsageDescription</key>
<string>This app needs access to your photo library to manage media files</string>
<key>NSPhotoLibraryAddUsageDescription</key>
<string>This app needs access to save media files to your photo library</string>

Permissions

Important: This plugin automatically shows native permission dialogs. No additional setup needed beyond AndroidManifest.xml and Info.plist configuration.

Basic Permission Flow

import { CapacitorMediaStore } from '@odion-cloud/capacitor-mediastore';

// Check current permissions
const current = await CapacitorMediaStore.checkPermissions();
console.log('Current permissions:', current);

// Request all permissions
const all = await CapacitorMediaStore.requestPermissions();
console.log('All permissions result:', all);

Granular Permission Types

// Request only audio permissions (music apps)
await CapacitorMediaStore.requestPermissions({ types: ['audio'] });

// Request images and video (gallery apps)
await CapacitorMediaStore.requestPermissions({ types: ['images', 'video'] });

// Request specific document access
await CapacitorMediaStore.requestPermissions({ types: ['document'] });

Permission States

State Description Action
granted Permission granted Can access media
denied Permission denied Show explanation or guide to settings
prompt Permission not requested Call requestPermissions()
prompt-with-rationale Previously denied Show rationale then request
Android Version Differences: Android 13+ requires separate permissions for images, audio, and video. Android 12 and below uses a single READ_EXTERNAL_STORAGE permission.

Usage

Basic Import

import { CapacitorMediaStore } from '@odion-cloud/capacitor-mediastore';

Get All Media Files

const result = await CapacitorMediaStore.getMedias({
  limit: 100,
  offset: 0,
  sortBy: 'DATE_ADDED',
  sortOrder: 'DESC',
  includeExternal: true // Include SD card
});

console.log('Total files:', result.totalCount);
console.log('Files:', result.media);

Get Media by Type

// Get audio files
const audioFiles = await CapacitorMediaStore.getMediasByType({
  mediaType: 'audio',
  sortBy: 'TITLE',
  sortOrder: 'ASC',
  includeExternal: true
});

// Get images
const images = await CapacitorMediaStore.getMediasByType({
  mediaType: 'image',
  limit: 50
});

// Get documents (PDF, DOC, etc.)
const documents = await CapacitorMediaStore.getMediasByType({
  mediaType: 'document'
});

Get Albums

const albums = await CapacitorMediaStore.getAlbums();
console.log('Albums found:', albums.albums.length);

Get Detailed Metadata

const metadata = await CapacitorMediaStore.getMediaMetadata({
  filePath: 'file:///storage/emulated/0/Music/song.mp3'
});

console.log('Song details:', metadata.media);
SD Card Support: Set includeExternal: true to access SD card files. This works across all Android versions.

API Reference

Methods

checkPermissions()

checkPermissions(): Promise<PermissionStatus>

Check current permission status without requesting.

requestPermissions()

requestPermissions(options?: RequestPermissionsOptions): Promise<PermissionStatus>

Request media permissions with native dialogs.

Options:
  • types?: string[] - Specific types: 'images', 'audio', 'video', 'document'

getMedias()

getMedias(options?: MediaQueryOptions): Promise<MediaResponse>

Get all media files from device storage.

getMediasByType()

getMediasByType(options: MediaTypeOptions): Promise<MediaResponse>

Get media files by specific type.

Media Types:
  • image - Photos, JPEG, PNG, etc.
  • audio - Music, MP3, AAC, etc.
  • video - Videos, MP4, AVI, etc.
  • document - PDF, DOC, TXT, etc.

getAlbums()

getAlbums(): Promise<AlbumResponse>

Get all music albums from device.

getMediaMetadata()

getMediaMetadata(options: MediaMetadataOptions): Promise<MediaMetadataResponse>

Get detailed metadata for a specific media file.

Interfaces

interface MediaQueryOptions {
  limit?: number;              // Limit number of results
  offset?: number;             // Pagination offset
  sortOrder?: 'ASC' | 'DESC';  // Sort direction
  sortBy?: string;             // Sort field
  albumName?: string;          // Filter by album
  artistName?: string;         // Filter by artist
  includeExternal?: boolean;   // Include SD card
}
interface MediaFile {
  id: string;
  uri: string;
  displayName: string;
  size: number;
  mimeType: string;
  dateAdded: number;
  dateModified: number;
  mediaType: string;
  width?: number;              // Images/videos
  height?: number;             // Images/videos
  duration?: number;           // Audio/video
  title?: string;              // Audio
  artist?: string;             // Audio
  album?: string;              // Audio
  isExternal?: boolean;        // SD card file
}

Examples

Music Player App

async function loadMusicLibrary() {
  // Request audio permissions
  await CapacitorMediaStore.requestPermissions({ types: ['audio'] });
  
  // Get all audio files
  const result = await CapacitorMediaStore.getMediasByType({
    mediaType: 'audio',
    sortBy: 'TITLE',
    sortOrder: 'ASC',
    includeExternal: true
  });
  
  // Filter SD card songs
  const sdCardSongs = result.media.filter(song => song.isExternal);
  console.log(`Found ${sdCardSongs.length} songs on SD card`);
  
  return result.media;
}

Photo Gallery App

async function loadPhotoGallery() {
  // Request image and video permissions
  await CapacitorMediaStore.requestPermissions({ 
    types: ['images', 'video'] 
  });
  
  // Get recent photos
  const photos = await CapacitorMediaStore.getMediasByType({
    mediaType: 'image',
    sortBy: 'DATE_ADDED',
    sortOrder: 'DESC',
    limit: 100
  });
  
  // Get recent videos
  const videos = await CapacitorMediaStore.getMediasByType({
    mediaType: 'video',
    sortBy: 'DATE_ADDED',
    sortOrder: 'DESC',
    limit: 50
  });
  
  return { photos: photos.media, videos: videos.media };
}

Document Manager App

async function loadDocuments() {
  // Request document permissions
  await CapacitorMediaStore.requestPermissions({ types: ['document'] });
  
  // Get document files (PDF, DOC, etc.)
  const documents = await CapacitorMediaStore.getMediasByType({
    mediaType: 'document',
    sortBy: 'DATE_MODIFIED',
    sortOrder: 'DESC'
  });
  
  // Filter by file type
  const pdfFiles = documents.media.filter(doc => 
    doc.mimeType === 'application/pdf'
  );
  
  return { allDocs: documents.media, pdfs: pdfFiles };
}

Complete Permission Setup

class MediaPermissionManager {
  async setupPermissions() {
    // Check current status
    const current = await CapacitorMediaStore.checkPermissions();
    
    if (this.hasRequiredPermissions(current)) {
      return { granted: true, permissions: current };
    }
    
    // Request all permissions
    const result = await CapacitorMediaStore.requestPermissions();
    
    return {
      granted: this.hasRequiredPermissions(result),
      permissions: result
    };
  }
  
  hasRequiredPermissions(permissions) {
    // Check for any granted permission
    return Object.values(permissions).some(state => state === 'granted');
  }
}

Troubleshooting

Permission Dialog Not Showing

Common Issues:
  • Missing permissions in AndroidManifest.xml
  • Missing usage descriptions in iOS Info.plist
  • Testing on simulator instead of real device
  • User already denied permissions permanently

Android-Specific Issues

Audio Files Not Found (Android 12 and below)

The plugin now uses MediaStore.Files.getContentUri() for Android 12 and below to ensure broader media access including audio files.

Document Files Not Filtering Correctly

Document type now properly filters for PDF, DOC, TXT files and excludes media files (audio, video, images).

Debug Permission Issues

async function debugPermissions() {
  console.log('Checking permissions...');
  const current = await CapacitorMediaStore.checkPermissions();
  console.log('Current permissions:', current);
  
  // Check each permission state
  Object.entries(current).forEach(([key, value]) => {
    console.log(`${key}: ${value}`);
    if (value === 'denied') {
      console.warn(`${key} is denied - user may need to enable in settings`);
    }
  });
}

Platform Differences

Platform Behavior Notes
Android 13+ Granular permissions Separate dialogs for images, audio, video
Android 6-12 Single storage permission READ_EXTERNAL_STORAGE covers all media
iOS Photo library permission Single permission for all media types
Web No permissions needed Development fallback only

Performance Tips

  • Use limit and offset for pagination
  • Set includeExternal: false if SD card access not needed
  • Use specific mediaType instead of getting all media
  • Cache permission results to avoid repeated checks
